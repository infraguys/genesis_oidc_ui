# Genesis OIDC UI

This repository contains a minimal React application bootstrapped with Vite and TypeScript.

## Getting started

### Prerequisites

- Node.js >= 18
- npm >= 9

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview

```bash
npm run preview
```

## Documentation

Project documentation is available in the `docs/` directory.
